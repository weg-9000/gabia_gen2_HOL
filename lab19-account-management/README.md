# Lab 21: 계정 관리

## 학습 목표

- 가비아 Gen2 계정 구조 이해 (메인 계정, 서브 계정, 그룹, 프로젝트)
- 서브 계정 생성 및 관리
- 프로젝트 생성 및 멤버 추가
- 역할 및 권한 부여

**소요 시간**: 30분
**난이도**: 초급

---

## 실습 단계

### 1. 콘솔 로그인

```
<https://console.gabiacloud.com/[그룹뒷자리주소]>

```

[입력 정보]

| 항목 | 값 |
| --- | --- |
| 아이디 | [메인 계정 ID] |
| 비밀번호 | [메인 계정 PW] |

로그인 후:

- 그룹 콘솔 진입
- 프로젝트 목록 확인

### 2. 프로젝트 생성

```
그룹 콘솔 > 프로젝트 > 생성

```

[입력 정보]

| 항목 | 값 |
| --- | --- |
| 프로젝트 이름 | lab-project |
| 설명 | Lab 실습용 프로젝트 |

제한 사항:

- 그룹당 최대 3개 프로젝트 생성 가능

생성 완료 후:

- 프로젝트 콘솔 접근 가능
- URL: [console.gabiacloud.com/[그룹뒷자리주소]/[project_id]/](http://console.gabiacloud.com/%5B%EA%B7%B8%EB%A3%B9%EB%92%B7%EC%9E%90%EB%A6%AC%EC%A3%BC%EC%86%8C%5D/%5Bproject_id%5D/)

### 3. 서브 계정 생성

```
그룹 콘솔 > 계정 관리 > 서브 계정 > 생성

```

[입력 정보]

| 항목 | 값 |
| --- | --- |
| 아이디 | lab-sub-user |
| 비밀번호 | [영문+숫자+특수문자 조합] |
| 이름 | 실습 사용자 |
| 이메일 | [user@example.com](mailto:user@example.com) |

서브 계정 특징:

- 메인 계정이 생성
- 가비아 계정과 매핑되지 않음
- 가비아 클라우드 Gen2에서만 사용

### 4. 프로젝트 멤버 추가

```
프로젝트 콘솔 > 멤버 관리 > 멤버 추가

```

[입력 정보]

| 항목 | 값 |
| --- | --- |
| 계정 선택 | lab-sub-user |
| 프로젝트 역할 | [역할 선택] |

멤버 추가 후:

- 해당 서브 계정은 추가된 프로젝트에만 접근 가능
- 부여된 프로젝트 역할에 따라 권한 제한

### 5. 역할 부여/변경

```
프로젝트 콘솔 > 멤버 관리 > [멤버 선택] > 역할 변경

```

[입력 정보]

| 항목 | 값 |
| --- | --- |
| 프로젝트 역할 | [변경할 역할 선택] |

역할 유형:

- 그룹 역할: 그룹 콘솔 내 권한
- 프로젝트 역할: 프로젝트 콘솔 내 권한

### 6. 서브 계정으로 로그인 테스트

```
<https://console.gabiacloud.com/[그룹뒷자리주소]>

```

[입력 정보]

| 항목 | 값 |
| --- | --- |
| 아이디 | lab-sub-user |
| 비밀번호 | [서브 계정 비밀번호] |

확인 사항:

- 멤버로 추가된 프로젝트만 접근 가능
- 부여된 역할에 따른 권한 제한 확인

### 7. 프로젝트 멤버 삭제

```
프로젝트 콘솔 > 멤버 관리 > [멤버 선택] > 삭제

```

삭제 후:

- 해당 서브 계정은 프로젝트 접근 불가

### 8. 서브 계정 삭제

```
그룹 콘솔 > 계정 관리 > 서브 계정 > [계정 선택] > 삭제

```

삭제 조건:

- 멤버로 추가된 모든 프로젝트에서 제거 필요

## 트러블슈팅

| 문제 | 원인 | 해결 |
| --- | --- | --- |
| 로그인 실패 | 5회 이상 실패 | 5분 후 재시도 |
| 로그인 불가 | 가비아 ID로 시도 | 클라우드 계정 ID 사용 |
| 프로젝트 생성 불가 | 3개 제한 초과 | 기존 프로젝트 삭제 후 생성 |
| 프로젝트 삭제 불가 | 리소스 존재 | 프로젝트 내 모든 리소스 삭제 후 진행 |
| 멤버 추가 불가 | 서브 계정 미생성 | 서브 계정 먼저 생성 |
| 권한 부족 | 역할 미부여 | 적절한 역할 부여 |
| 세션 만료 | 3시간 초과 | 재로그인 |
| 계정 정지 | 연체 정지 상태 | 미납 요금 결제 |

## 참고: 계정 구조 다이어그램

```
가비아 계정 (Gabia ID)
    │
    └── 메인 계정 (Main Account) ─────┬── 그룹 (Group)
        │                             │
        │                             ├── 프로젝트 A ── 리소스
        │                             │       └── 멤버 (서브계정+역할)
        │                             │
        │                             ├── 프로젝트 B ── 리소스
        │                             │       └── 멤버 (서브계정+역할)
        │                             │
        │                             └── 프로젝트 C ── 리소스 (최대 3개)
        │                                     └── 멤버 (서브계정+역할)
        │
        └── 서브 계정 1 (Sub Account)
        └── 서브 계정 2 (Sub Account)
        └── 서브 계정 N (Sub Account)

```
